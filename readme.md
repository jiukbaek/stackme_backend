# Stack Me
<img src="./static/image/logo.png" style="width:80px;"/><br/>
https://stackme.co.kr<br/>
**스택 미**는 자신의 경력, 프로젝트를 관리하는 웹 서비스입니다.

## 사용기술

### Client
* React
* SCSS
* Redux
  
### Server
* Node(Express)
* Webpack
* Babel
* MySQL
* Sequelize
* AWS EC2
* Docker

## 특징
 * JWT를 이용한 로그인 처리
 * Express middleware를 통한 API_KEY 처리
