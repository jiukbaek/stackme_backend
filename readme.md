# Stack Me
<img src="./static/image/logo.png" width="80"/><br/>
https://stackme.co.kr<br/>
**스택 미**는 자신의 경력, 프로젝트를 관리하는 웹 서비스입니다.

## 사용기술

### Client
* React
* SCSS
* Redux
  
### Server
* Node(Express)
* MySQL
* Sequelize

### DevOps
* Webpack
* Babel
* AWS EC2
* Docker


## 특징
 * bcrypt를 이용한 유저 패스워드 해쉬 처리
 * jwt, passport.js를 이용한 로그인 처리
